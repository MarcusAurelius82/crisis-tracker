# 🌍 Crisis Tracker

A real-time global humanitarian crisis monitoring dashboard. Visualizes active disasters and emergency events on an interactive 3D globe, pulling live data from GDACS and ReliefWeb.

---

## Features

- **Interactive 3D Globe** — drag, zoom, and explore crisis markers worldwide
- **Live Data** — syncs every 5 minutes from GDACS and ReliefWeb
- **Severity Filtering** — filter by RED / ORANGE alert level or crisis type
- **HUD UI** — tactical dark/light mode with system status indicators
- **Simulation Fallback** — displays mock data if live feeds are unavailable
- **Fully Responsive** — mobile and desktop layouts

---

## Data Sources

| Source | Data |
|--------|------|
| [GDACS](https://www.gdacs.org) | Earthquakes, floods, cyclones, volcanoes, droughts |
| [ReliefWeb](https://reliefweb.int) | Humanitarian disasters and emergency reports |
